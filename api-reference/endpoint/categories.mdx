---
title: "Get All Categories"
description: "Retrieves a list of all categories in a workspace."
---

<RequestExample>
```bash cURL
curl "https://api.marblecms.com/cm6ytuq9x0000i803v0isidst/categories"
```

```javascript Node.js
const fetch = require("node-fetch");

fetch("https://api.marblecms.com/cm6ytuq9x0000i803v0isidst/categories")
  .then((res) => res.json())
  .then((json) => console.log(json))
  .catch((err) => console.error("error:" + err));
```

</RequestExample>
