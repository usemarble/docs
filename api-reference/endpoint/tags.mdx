---
title: "Get All Tags"
description: "Retrieves a list of all tags in a workspace."
---

<RequestExample>
```bash cURL
curl "https://api.marblecms.com/cm6ytuq9x0000i803v0isidst/tags"
```

```javascript Node.js
const fetch = require("node-fetch");

fetch("https://api.marblecms.com/cm6ytuq9x0000i803v0isidst/tags")
  .then((res) => res.json())
  .then((json) => console.log(json))
  .catch((err) => console.error("error:" + err));
```

</RequestExample>
