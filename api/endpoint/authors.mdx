---
title: "Get All Authors"
description: "Retrieves a paginated list of all authors in your workspace."
api: "GET /v1/authors"
---

<RequestExample>
```bash cURL
curl -H "Authorization: Bearer YOUR_API_KEY" "https://api.marblecms.com/v1/authors"
```

```javascript JavaScript
const response = await fetch("https://api.marblecms.com/v1/authors", {
  headers: { Authorization: "Bearer YOUR_API_KEY" },
});
const data = await response.json();
console.log(data.authors);
```

</RequestExample>

## Authorizations

<ParamField header="Authorization" type="string" required>
  Your Marble API key. Include it in the header as: `Bearer YOUR_API_KEY`
</ParamField>

## Query Parameters

<ParamField query="limit" type="integer" default="10">
  Maximum number of authors per page. Range: `1-100`.
</ParamField>

<ParamField query="page" type="integer" default="1">
  Page number to return.
</ParamField>

<Note>
  Responses include a `pagination` object with navigation helpers. For advanced
  usage and error handling, see the [Pagination guide](/api/pagination).
</Note>
