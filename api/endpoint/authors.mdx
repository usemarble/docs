## Get a Single Author

Retrieve a specific author by their slug or ID, optionally including their posts.

```
GET /v1/authors/:identifier
```

<RequestExample>
```bash cURL
curl -H "Authorization: YOUR_API_KEY" \
  "https://api.marblecms.com/v1/authors/john-doe"
```

```bash cURL (with posts)
curl -H "Authorization: YOUR_API_KEY" \
  "https://api.marblecms.com/v1/authors/john-doe?include=posts"
```

</RequestExample>

### Path Parameters

<ParamField path="identifier" type="string" required>
  The author slug (e.g., `john-doe`) or ID.
</ParamField>

### Query Parameters

<ParamField query="include" type="string">
  Include related data. Use `posts` to include the author's posts. ```
  ?include=posts ```
</ParamField>

<ParamField query="limit" type="integer" default="20">
  When `include=posts`, the maximum number of posts to return.
</ParamField>

<ParamField query="page" type="integer" default="1">
  When `include=posts`, the page of posts to return.
</ParamField>

| X (Twitter) | `x`         |

| Instagram   | `instagram` |
| YouTube     | `youtube`   |
| TikTok      | `tiktok`    |
| Discord     | `discord`   |
| Bluesky     | `bluesky`   |
| Website     | `website`   |
