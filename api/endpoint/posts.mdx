## Query Parameters

Use query parameters to paginate, sort, and filter results.

### Pagination

<ParamField query="limit" type='integer | "all"' default="10">
  Maximum number of items per page. - Use a number between `1` and `200` - Use
  `"all"` to return all posts in a single response (use with caution for large
  datasets)
</ParamField>

<ParamField query="page" type="integer" default="1">
  Page number to return. Pages start at 1.
</ParamField>

### Sorting

<ParamField query="order" type="string" default="desc">
  Sort order by `publishedAt` date. **Accepted values:** `asc`, `desc`
</ParamField>

### Filtering

<ParamField query="categories" type="string">
  Comma-separated list of category slugs to include. ```
  ?categories=tutorials,announcements ```
</ParamField>

<ParamField query="excludeCategories" type="string">
  Comma-separated list of category slugs to exclude. ```
  ?excludeCategories=drafts,internal ```
</ParamField>

<ParamField query="tags" type="string">
  Comma-separated list of tag slugs. Returns posts matching any of the specified
  tags. ``` ?tags=product,engineering ```
</ParamField>

<ParamField query="excludeTags" type="string">
  Comma-separated list of tag slugs to exclude. ```
  ?excludeTags=archived,deprecated ```
</ParamField>

<ParamField query="query" type="string">
  Full-text search across `title` and `content` fields. ``` ?query=getting
  started ```
</ParamField>

### Content Format

<ParamField query="format" type="string">
  Format of the `content` field in the response. - **Default:** HTML -
  **`markdown`:** Returns content converted to Markdown ``` ?format=markdown ```
</ParamField>

## Examples

<AccordionGroup>
<Accordion title="Filter by category">
```bash
curl -H "Authorization: YOUR_API_KEY" \
  "https://api.marblecms.com/v1/posts?categories=tutorials"
```
</Accordion>

<Accordion title="Multiple filters">
  ```bash curl -H "Authorization: YOUR_API_KEY" \
  "https://api.marblecms.com/v1/posts?categories=tutorials&tags=beginner&limit=5"
  ```
</Accordion>

<Accordion title="Get all posts as Markdown">
  ```bash curl -H "Authorization: YOUR_API_KEY" \
  "https://api.marblecms.com/v1/posts?limit=all&format=markdown" ```
</Accordion>

<Accordion title="Search posts">
```bash
curl -H "Authorization: YOUR_API_KEY" \
  "https://api.marblecms.com/v1/posts?query=getting%20started"
```
</Accordion>
</AccordionGroup>
