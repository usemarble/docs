## Get a Single Tag

Retrieve a specific tag by its slug or ID, optionally including associated posts.

```
GET /v1/tags/:identifier
```

<RequestExample>
```bash cURL
curl -H "Authorization: YOUR_API_KEY" \
  "https://api.marblecms.com/v1/tags/product"
```

```bash cURL (with posts)
curl -H "Authorization: YOUR_API_KEY" \
  "https://api.marblecms.com/v1/tags/product?include=posts"
```

</RequestExample>

### Path Parameters

<ParamField path="identifier" type="string" required>
  The tag slug (e.g., `product`) or ID.
</ParamField>

### Query Parameters

<ParamField query="include" type="string">
  Include related data. Use `posts` to include posts with this tag. ```
  ?include=posts ```
</ParamField>

<ParamField query="limit" type="integer" default="20">
  When `include=posts`, the maximum number of posts to return.
</ParamField>

<ParamField query="page" type="integer" default="1">
  When `include=posts`, the page of posts to return.
</ParamField>
