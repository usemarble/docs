---
title: "Get All Tags"
description: "Retrieves a paginated list of all tags in your workspace."
api: "GET /v1/tags"
---

<RequestExample>
```bash cURL
curl -H "Authorization: YOUR_API_KEY" "https://api.marblecms.com/v1/tags"
```

```javascript JavaScript
const response = await fetch("https://api.marblecms.com/v1/tags", {
  headers: { Authorization: "YOUR_API_KEY" },
});
const data = await response.json();
console.log(data.tags);
```

```python Python
import requests

response = requests.get(
    "https://api.marblecms.com/v1/tags",
    headers={"Authorization": "YOUR_API_KEY"}
)
print(response.json())
```

</RequestExample>

## Query Parameters

<ParamField query="limit" type="integer" default="10">
  Maximum number of tags per page. Range: `1-100`.
</ParamField>

<ParamField query="page" type="integer" default="1">
  Page number to return.
</ParamField>

<Note>
  Responses include a `pagination` object with navigation helpers. For advanced
  usage and error handling, see the [Pagination guide](/api/pagination).
</Note>
