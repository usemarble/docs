---
title: "Get a Single Author"
description: "Retrieves a single author by their identifier (slug or id)."
api: "GET /v1/authors/:identifier"
---

<RequestExample>
```bash cURL
curl -H "Authorization: Bearer YOUR_API_KEY" \
  "https://api.marblecms.com/v1/authors/john-doe"
```

```javascript JavaScript
const response = await fetch("https://api.marblecms.com/v1/authors/john-doe", {
  headers: { Authorization: "Bearer YOUR_API_KEY" },
});
const data = await response.json();
console.log(data);
```

</RequestExample>

## Authorizations

<ParamField header="Authorization" type="string" required>
  Your Marble API key. Include it in the header as: `Bearer YOUR_API_KEY`
</ParamField>

### Path Parameters

<ParamField path="identifier" type="string" required>
  The author slug (e.g., `john-doe`) or ID.
</ParamField>

### Query Parameters

<ParamField query="include" type="string">
  Include related data. Use `posts` to include the author's posts.
</ParamField>

<ParamField query="limit" type="integer" default="20">
  When `include=posts`, the maximum number of posts to return.
</ParamField>

<ParamField query="page" type="integer" default="1">
  When `include=posts`, the page of posts to return.
</ParamField>
