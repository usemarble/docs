---
title: "Get a Single Author"
description: "Retrieves a single author by their identifier (slug or id)."
api: "GET /v1/authors/:identifier"
---

<RequestExample>
```bash cURL
curl -H "Authorization: Bearer YOUR_API_KEY" \
  "https://api.marblecms.com/v1/authors/john-doe"
```

```javascript JavaScript
const response = await fetch("https://api.marblecms.com/v1/authors/john-doe", {
  headers: { Authorization: "Bearer YOUR_API_KEY" },
});
const data = await response.json();
console.log(data);
```

</RequestExample>

<ResponseExample>
```json Response
{
  "id": "clt92f96m000008l4gken6v2t",

  "slug": "john-doe",

```
</ResponseExample>

## Authorizations

<ParamField header="Authorization" type="string" required>
  Your Marble API key. Include it in the header as: `Bearer YOUR_API_KEY`
</ParamField>

### Path Parameters

<ParamField path="identifier" type="string" required>
  The author ID or slug.
</ParamField>

### Query Parameters

<Note>
  This endpoint does not return posts. Use the [posts
  endpoint](/api/endpoint/posts) with the `author` filter to retrieve posts by
  author.
</Note>
