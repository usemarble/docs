---
title: "Get a Single Category"
description: "Retrieves a single category by its identifier (slug or id)."
api: "GET /v1/categories/:identifier"
---

<RequestExample>
```bash cURL
curl -H "Authorization: Bearer YOUR_API_KEY" \
  "https://api.marblecms.com/v1/categories/tutorials"
```

```javascript JavaScript
const response = await fetch(
  "https://api.marblecms.com/v1/categories/tutorials",
  {
    headers: { Authorization: "Bearer YOUR_API_KEY" },
  }
);
const data = await response.json();
console.log(data);
```

</RequestExample>

<ResponseExample>
```json Response
{
  "id": "clt92hrnp000308l4ajp74b32",
  "name": "Tutorials",
  "slug": "tutorials",
  "description": null,
  "count": {
    "posts": 12
  }
}
```
</ResponseExample>

## Authorizations

<ParamField header="Authorization" type="string" required>
  Your Marble API key. Include it in the header as: `Bearer YOUR_API_KEY`
</ParamField>

### Path Parameters

<ParamField path="identifier" type="string" required>
  The category ID or slug.
</ParamField>

### Query Parameters

<Note>
  This endpoint does not return posts. Use the [posts
  endpoint](/api/endpoint/posts) with the `category` filter to retrieve posts by
  category.
</Note>
