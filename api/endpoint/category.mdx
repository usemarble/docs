---
title: "Get a Single Category"
description: "Retrieves a single category by its identifier (slug or id)."
api: "GET /v1/categories/:identifier"
---

<RequestExample>
```bash cURL
curl -H "Authorization: YOUR_API_KEY" \
  "https://api.marblecms.com/v1/categories/tutorials"
```

```bash cURL (with posts)
curl -H "Authorization: YOUR_API_KEY" \
  "https://api.marblecms.com/v1/categories/tutorials?include=posts"
```

</RequestExample>

### Path Parameters

<ParamField path="identifier" type="string" required>
  The category slug (e.g., `tutorials`) or ID.
</ParamField>

### Query Parameters

<ParamField query="include" type="string">
  Include related data. Use `posts` to include posts in this category.
</ParamField>

<ParamField query="limit" type="integer" default="20">
  When `include=posts`, the maximum number of posts to return.
</ParamField>

<ParamField query="page" type="integer" default="1">
  When `include=posts`, the page of posts to return.
</ParamField>
