---
title: "Get a Single Post"
description: "Retrieves a single post by its identifier (slug or id)."
---

<RequestExample>
```bash cURL (by slug)
curl "https://api.marblecms.com/v1/cm6ytuq9x0000i803v0isidst/posts/introducing-marble"
```

```bash cURL (by id)
curl "https://api.marblecms.com/v1/cm6ytuq9x0000i803v0isidst/posts/clt92n38p000108l48zkj41an"
```

```bash cURL (with markdown format)
curl "https://api.marblecms.com/v1/cm6ytuq9x0000i803v0isidst/posts/introducing-marble?format=markdown"
```

```javascript Node.js
const fetch = require("node-fetch");

// You can pass either a slug or an id as the identifier
const identifier = "introducing-marble"; // or: "clt92n38p000108l48zkj41an"

fetch(
  `https://api.marblecms.com/v1/cm6ytuq9x0000i803v0isidst/posts/${identifier}`
)
  .then((res) => res.json())
  .then((json) => console.log(json))
  .catch((err) => console.error("error:" + err));
```

</RequestExample>

### Path Parameters

<ParamField path="identifier" type="string" required>
  The slug or id of the post to retrieve.
</ParamField>

### Query Parameters

<ParamField query="format" type="string">
  The format of the content field in the response. Set to `markdown` to return the content as markdown instead of HTML.
</ParamField>
